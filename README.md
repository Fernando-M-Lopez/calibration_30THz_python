# calibration_30THz_python
This repository contains the calibration routines created to be used with the Mid-IR telescopes AR30T and SP30T
- seq2fits_30THz.py must be used to extract .fits images from .seq files during normal observation
- seq2fits_30THz_flat_frames.py must be used to extract .fits images from .seq files during flat adquisition (based in                                  method KLL)
